# WhatsApp Bot Simulator

A web-based WhatsApp bot simulator with DOGMEN pairing code.

## Features
- Pairing code: DOGMEN
- All bot commands (.movie, .yt, .gg, .tt, .ping, .menu)
- WhatsApp-like interface
- Mobile responsive

## Deploy on Vercel
1. Push to GitHub
2. Import on Vercel
3. Deploy automatically
